# tracker-api

## Unit Test

```
./vendor/bin/phpunit ./tests/UserTest.php
```

## Create a local server on PHP

```
php -S 127.0.0.1:8080 -t public
```

## autoload update command. Run it anytime you change _composer.json_

```
composer dump-autoload
```
